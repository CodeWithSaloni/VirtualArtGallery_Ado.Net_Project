# Virtual Art Gallery Management System

## Overview

Virtual Art Gallery Management System is a desktop application developed using C#, Windows Forms, ADO.NET, and SQL Server. The application allows users to manage artwork records efficiently through Create, Read, Update, and Delete (CRUD) operations.

## Features

* User Registration and Login
* Add New Artwork
* View Artwork Details
* Update Existing Artwork Information
* Delete Artwork Records
* Search and Manage Artwork Data
* Database Connectivity using ADO.NET
* Data Handling using DataSets

## Technologies Used

* C#
* Windows Forms
* ADO.NET
* SQL Server
* Visual Studio

## Artwork Details Managed

* Art ID
* Title
* Artist Name
* Year of Publication
* Description
